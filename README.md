# School Bus Route Manager

## Overview
A simple Python-based CRUD application to manage school bus routes, including route creation, updating, deletion, and viewing.

## Features
- Add new bus routes with stops
- View all existing routes
- Update existing route details
- Delete routes by ID
- Simple console-based interface

## Technologies / Tools Used
- Python 3.x
